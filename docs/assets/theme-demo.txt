hat 0.1.0
A silly help message to demo theme support.

Usage: hat [options] [arguments] [subcommand]

Options:
  -h, --help           Show this help message and exit
  -t, --top-color [COLOR]  Set the top color of the hat (default: "magenta")
  -b, --bottom-color [COLOR]  Set the bottom color of the hat (default: "lime green")
  -s, --size [SIZE]    Set the size of the hat (default: "bigger than your head")
  +w, --wizard         Turn the hat into a wizard hat

Arguments:
  <HAT-NAME>  The name of the hat
  <HAT-TYPE>...  One or more types for the hat

Subcommands:
  create    Create a new hat
  modify    Modify an existing hat
  delete    Delete an existing hat

Other:
  ~/.config/hat/config  Location of your config file

Note: This command is not to be taken seriously. Use at your own risk.
